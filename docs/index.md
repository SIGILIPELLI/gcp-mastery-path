# GCP Mastery Path

A structured, module-wise Google Cloud Platform training program that takes
you from creating your first free-tier project to designing production-grade,
master-level cloud architecture — with runnable `gcloud` CLI commands and
config in every module and a hands-on project at the end of each level.

## How the program is organized

| Level | Focus | Modules |
|-------|-------|---------|
| [Level 1 · Entry](level-1/index.md) | `gcloud` CLI, IAM, Compute Engine, Cloud Storage, networking, Cloud SQL, serverless, monitoring, IaC | 9 topics + 1 capstone |
| [Level 2 · Intermediate](level-2/index.md) | GKE, autoscaling & load balancing, Cloud DNS, Pub/Sub, Firestore/Bigtable, App Engine, cost management | 9 topics + 1 project |
| [Level 3 · Advanced](level-3/index.md) | Advanced networking, GKE at depth, Cloud Workflows, advanced IAM, Terraform, CI/CD, security | 9 topics + 1 project |
| [Level 4 · Master](level-4/index.md) | Architecture framework, data analytics (BigQuery), GKE at scale, multi-project strategy, compliance | 9 topics + 1 capstone |

## How to use this site

- Work through each level in order — later modules assume earlier ones.
- Every topic page has runnable `gcloud` commands and config snippets — try
  them in a real (free-tier) GCP project, or in
  [Cloud Shell](https://cloud.google.com/shell) which needs no local install.
- Each level ends with a project that combines everything learned in that level.
- Use the search bar (top of the page) to jump straight to a topic.
- Level 1's capstone includes explicit teardown steps — always clean up
  resources when you're done practicing so a free-tier project doesn't
  accidentally start incurring charges.

Start here → [Level 1 · Entry](level-1/index.md)
