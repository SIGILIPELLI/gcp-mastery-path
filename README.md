# GCP Mastery Path

A free, structured, module-wise Google Cloud Platform training program —
entry level to master level, with runnable `gcloud` CLI commands and config
in every module and a hands-on project at the end of each level.

**Live site:** https://sigilipelli.github.io/gcp-mastery-path/

## Contents

- **Level 1 · Entry** — `gcloud` CLI, IAM & security basics, Compute Engine, Cloud Storage, VPC networking, Cloud SQL, Cloud Functions/Cloud Run, monitoring & logging, Terraform intro, capstone project
- **Level 2 · Intermediate** — GKE basics, autoscaling & load balancing, Cloud DNS, Pub/Sub, Firestore/Bigtable, App Engine, Secret Manager, Cloud CDN & Armor, cost management
- **Level 3 · Advanced** — advanced VPC, GKE advanced, Cloud Workflows, advanced IAM, Terraform on GCP, CI/CD (Cloud Build), multi-region DR, advanced monitoring, security deep dive
- **Level 4 · Master** — architecture framework, advanced serverless (Eventarc), BigQuery/Dataflow/Dataproc, GKE at scale, multi-project strategy, cost optimization, advanced networking, compliance & governance, chaos engineering, capstone

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
